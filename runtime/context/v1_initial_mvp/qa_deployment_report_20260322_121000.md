## 2. CI/CD Pipeline Strategy (GitHub Actions)

To deploy this framework, the following GitHub Actions pipeline structure is recommended:

1.  **Job 1: Test & Lint**
    -   Run PHPUnit.
    -   Run ESLint and React component tests.
    -   Validate Terraform configuration (`terraform validate`).
2.  **Job 2: Build & Push Container**
    -   Build Docker image for the Laravel/React monolith.
    -   Push image to Google Container Registry (GCR): `gcr.io/merkle-taskflow-ai-prod/taskflow-app:latest`
3.  **Job 3: Infrastructure Apply (Requires Manual Approval)**
    -   Execute `terraform apply -auto-approve` based on `output/infra/main.tf`.
4.  **Job 4: Deploy App**
    -   Trigger Cloud Run to deploy the latest container image.
    -   Run Laravel Migrations via Cloud Run jobs or a managed startup script.
