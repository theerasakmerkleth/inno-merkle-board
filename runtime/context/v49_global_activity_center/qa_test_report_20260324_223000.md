**Date:** 2026-03-24 22:30:00

## 🎯 Test Objectives
Validate that the centralized activity log is accessible, correctly filtered by user permissions, and free of 404/500 errors.

## 🛠 Bug Fixes during QA
1.  **Issue:** 404 Error when visiting `/activity`.
    *   **Cause:** Route Cache was outdated.
    *   **Fix:** Executed `php artisan route:clear`.
2.  **Issue:** 500 SQL Error (`ambiguous column name: id`).
    *   **Cause:** JOIN query in `ActivityLogController` between `projects` and `project_user` had overlapping `id` columns.
    *   **Fix:** Explicitly selected `projects.id`.
