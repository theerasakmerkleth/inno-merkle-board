## 1. UX/UI Vision Alignment

Continuing our "Swiss Minimalist" (Minimal Enterprise Grade) journey, we are expanding the global application shell to accommodate cross-project visibility and in-task collaboration.

---

## 2. Structural Redesign (Wireframes)

### 2.1 Global Navigation (Left Sidebar Updates)
The sidebar must now prioritize global views before diving into specific projects.
*   **Top Area (Workspace):**
    *   `My Tasks` (Icon: `inbox`) - The new default landing page (`/`).
    *   `Analytics Dashboard` (Icon: `bar_chart`) - Visible based on global permissions (Admin/PM).
    *   `Directory` (Icon: `group`) - Moved up from the project-specific area to the global workspace area, as users are managed globally.
*   **Middle Area (Projects):**
    *   The `Project Switcher` dropdown remains.
    *   Once a project is selected, the sub-navigation (`Boards`, `Project Settings`) appears below it.

### 2.2 My Tasks View (`/my-tasks` or `/`)
*   **Layout:** A high-density, edge-to-edge data table or a simplified vertical list.
*   **Columns:** `ID` (e.g., `[CORE-142]`), `Title`, `Project`, `Priority`, `Status`, `Due Date` (if applicable).
*   **Interaction:** Clicking a row opens the standard `TaskModal` overlay, allowing the user to update the status or add a comment without leaving the Inbox context.

### 2.3 Enterprise Analytics Dashboard (`/analytics`)
*   **Layout:** A standard grid of `shadcn/Card` components.
*   **Card 1: Resource Loading (Heatmap/List):** A table listing team members and a horizontal stacked bar chart (`shadcn/Progress` or simple HTML `div` bars) representing their current workload (`To Do` vs `In Progress`).
*   **Card 2: Velocity & Cycle Time:** Big number displays (e.g., "Avg. Cycle Time: 2.4 Days") with a subtle sparkline or trend indicator below it.

### 2.4 Task Modal Updates (Comments)
*   **Layout:** Inside the existing `TaskModal`, append a vertical timeline section below the main description.
*   **Comment Input:** A minimal `<textarea>` with a "Post" button (`variant="ghost"` or `outline`).
*   **Comment Display:** 
    *   A vertical line connecting comments.
    *   Small Avatar (`h-6 w-6`).
    *   Name and relative timestamp (`text-zinc-500 text-[10px]`).
    *   The comment text itself. If the author is an `AI Agent`, apply a subtle background tint (`bg-indigo-950/20`) and an AI icon to distinguish it from human comments.
